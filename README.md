# Yadro_BS
Репозиторий представляет собой код, реализующий систему, которая следит за работой комп. клуба согласно тестовому заданию. Программа выводит сообщения на английском языке чтобы избежать возможных ошибок с кодировкой. Club.exe запускает программу, в test.txt находится тестовый набор событий.
